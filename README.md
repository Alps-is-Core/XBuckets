# 𝕏 Buckets

**𝕏 Buckets** is a premium Chrome extension designed for power users of X (formerly Twitter). It provides a seamless, gesture-based way to organize the content you care about—posts, comments, and XChat messages—into custom, categorized "buckets."

No more losing track of interesting threads or important messages. Just long-press, drag, and drop.

![X Buckets Interface](https://raw.githubusercontent.com/username/XBuckets/main/icons/icon128.png)

## ✨ Features

- **Gesture-Based Saving**: Long-press (600ms) on any post or message to activate the "Bucket Overlay."
- **Edge-Based UI**: Buckets appear dynamically on the edges of your screen (Left, Right, and Bottom) for a fast, intuitive drag-and-drop experience.
- **Smart Extraction**: 
    - **Posts & Comments**: Saves full permalinks, text, author details, and media.
    - **XChat Messages**: Extracts full message content, including images, videos, and shared posts from your DMs.
- **Premium Organization**: Create, rename, and customize buckets with unique emojis and colors.
- **Viewer Panel**: A dedicated side-panel viewer with full-text search and tab-based filtering to browse your saved content without leaving X.
- **Privacy First**: All data is stored locally in your browser. No external servers, no tracking.

## 🚀 Getting Started

### Installation

Since this is currently in development, you can install it as an "Unpacked Extension":

1.  Download or clone this repository to your local machine.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** using the toggle in the top-right corner.
4.  Click the **Load unpacked** button.
5.  Select the `XBuckets` folder from your machine.

### How to Use

1.  Click the **𝕏 Buckets** icon in your extension bar to create your first few buckets (e.g., "Inspiration," "Stocks," "Memes").
2.  Navigate to [x.com](https://x.com).
3.  **Long-press** (hold for about 0.6s) on any post or DM.
4.  The "Bucket Overlay" will appear. **Drag** the item to one of the tabs on the screen edges.
5.  **Release** to save.
6.  Use the Side Panel (via the extension icon) to search and browse your saved items.
